# 单元测试方案

## 对象面板测试

* 使用模拟器ioSimu仿真设备，连接TDS服务

* TDS服务完成配置后，在"监控列表"页面打开对象面板

* 观察对象面板的实时数据和通道数据是否一致

* 读取并配置参数，观察与模拟器上的参数是否一致

* 使用真实硬件连接TDS

* 观察对象面板的实时数据和通道数据是否一致

* 读取并配置参数，观察与模拟器上的参数是否一致
